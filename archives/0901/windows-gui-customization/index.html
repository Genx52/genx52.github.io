1.桌面美化

要在 Windows 上实现类似第一张桌面图片的效果需要使用 Rainmeter 软件实现。（点击软件名即可进入官网）
完成下载和安装后，我们便可以使用 Rainmeter 美化桌面了。如果需要实现类似第一张图片的效果，请选择 Bohemo 皮肤。

Rainmeter的主题配置是十分简单的。绝大部分 Rainmeter 主题都已被打包为 .rmskin 一键安装包，只要轻点一下，便可开始安装。

完成安装后，Rainmeter 会自动应用当前安装的皮肤，并且显示出可用的控件。你可以自由拖动这些控件到适当的位置，然后锁定并保存控件位置。

使用 Rainmeter，你可以完全由主题内容以及你自己的喜好来美化你的 Windows 桌面。除 Rainmeter 外，还有 AIMP 等工具也可用于美化桌面。接下来，我会带来一些其他的系统美化内容，以及介绍一些我个人推荐的其它计算机软件。

2.任务栏、开始菜单和主题设置

这里简单介绍使用 StartIsBack++ 软件进行深度任务栏个性化的方法。

程序官网：http://www.startisback.com

上图中使用了 StartIsBack++ 的“自定义任务栏颜色（全透明）”、“任务栏图标居中”等功能，并使用了自定义开始菜单（原为“开始”屏幕）。其中，Windows 10 以下的操作系统因为不适配可将“自定义任务栏颜色”透明度调整为 0% 及设置“任务栏图标居中”的新版 StartIsBack++，所以使用 Windows 8/8.1、Windows 7等系统的用户将无法实现“全透明任务栏”、“任务栏图标居中”等功能。

2.3 配置图标包

其实一部分 Windows 主题已经自带图标包。但如果你对主题自带的图标也不满意，你也可以选择使用第三方制作的独立图标包。
** 第三方制作的图标包分为多种类型，其中最常见的是 iPack 一键安装包和 .ico 格式图标文件。以下将介绍 iPack 图标和 .ico格式图标的安装方式。在开始配置前，请务必对系统默认图标文件进行备份，以便将来恢复。

2.3.1 配置 iPack 图标包

为避免安装过程中可能导致的系统错误所带来的风险，安装前务必备份系统及安装硬盘版WinPE系统；开始安装 iPack 图标包前必须卸载或退出360、金山毒霸等杀毒软件；在安装新图标包前必须卸载之前安装过的图标包恢复到系统默认；Win10在安装图标包时仅只能替换imageres.dll文件，在安装时请选择Patch only selected files，且仅勾选imageres.dll。
** Win10 1903只能安装一个图标包，安装第二个图标包前必须卸载已经安装的图标包！

iPack 图标包开始安装后，Windows 资源管理器（包括桌面）将会自动关闭，安装完成后，部分图标效果已经显示出来，部分图标重启后将会生效。

请不要在安装过程中结束安装程序进程，否则将会导致未来安装的图标均不会生效。

2.3.2 配置 .ico 图标

操作方式很简单，对于快捷方式，鼠标在快捷方式图标上右键，然后定位到“属性”（Properties）选项：

3. 配置系统字体

说实话，我已经在电脑上看了将近 4 年的微软雅黑字体了，所以我打算找一款软件修改我的电脑的默认显示字体。后来我找到了 noMeiryoUI 字体修改软件和 MacType 字体渲染软件。经过一番修改，我得到了令我满意的效果：

** ↑ 修改前

** ↑ 修改后
修改后的字体比以前更加匀称，显示效果也更为柔和。
noMeiryoUI 的使用方法十分简单，只需要使用“Select”项选择自己理想的字体，并使用“Set”设置，即可替换系统默认的显示字体。

至于字体渲染软件 MacType，它是一个开源项目，能接管 Windows 系统的 GDI 字体渲染功能，实现比Mac系统更华丽的字体渲染效果。MacType 支持 WinXP、Win7、Win8、Win10 等操作系统。安装过程十分简单，到最后，它才会让我们在四种加载方式之间进行选择，对于一般用户来说，最好选择MacTray方式加载（MacTray也有两种方式，建议采用独立加载模式）。这种方式会在系统托盘中显示一个控制图标，方便配置。

4. 其他软件、站点推荐

单纯的对 Windows 用户界面进行个性化操作并不意味着就可以优雅地上网冲浪了。因为总是有一些捆绑垃圾软件、广告病毒之类的流氓软件出现在网络上。这里是我常用的一些高质量的 PC 应用软件以及网站，我也想分享给你们。

4.1 支持多语言的解压缩软件 Bandizip

我觉得 Bandizip 作为解压缩最大的优势就是可以自由选择解压时使用的文字编码了，其次才是无广告（WinRAR中文版出来挨打）。
如果你使用一般的解压缩软件来解压使用特定编码压缩的文件，那么就是这个效果：

通常情况下，这样解压出来的文件都是乱码，不能被正常读取。
而 Bandizip 可以自由选择文件解压时所使用的编码类型，以确保解压得到的文件不会因为乱码而影响质量。

官网：http://www.bandisoft.com/

4.2 动态壁纸软件 Wallpaper Engine（付费共享软件）


Wallpaper Engine 使您可以在Windows桌面上使用动态壁纸。它支持各种类型的动画壁纸，包括3D和2D动画、网站、视频甚至某些应用程序。既可选择现有壁纸，也可创建自己的壁纸并在 Steam 创意工坊上分享！它通过实时图形、视频、应用程序或网站让您的桌面充满活力，使用您喜欢的颜色实现动画壁纸个性化。Wallpaper Engine 使用可通过鼠标控制的交互式壁纸。支持多种宽高比和原始分辨率，包括 16:9、21:9、16:10、4:3。Wallpaper Engine 也支持多显示器环境。且壁纸默认将在玩游戏时暂停，以节省性能。正版用户可以在 Steam 创意工坊上分享和下载壁纸。Wallpaper Engine 可以与任何其他 Steam 游戏或应用程序同时使用。
Wallpaper Engine 支持的视频格式：mp4、WebM、avi、m4v、mov、wmv（对于本地文件，Workshop 仅支持 mp4）。
Wallpaper Engine 支持 Corsair iCUE。
如果感觉不错请在Steam购买正版，价格为 19 元（原价 18 元，由于汇率原因而涨价）。
购买链接：https://store.steampowered.com/app/431960/Wallpaper_Engine/
